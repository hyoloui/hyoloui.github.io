# 마크다운 기능

도큐사우르스는 **[Markdown](https://daringfireball.net/projects/markdown/syntax)** 기능과 몇가지 **추가기능을 지원**합니다.

## Front Matter

마크다운 문서 상단에 [Front Matter](https://jekyllrb.com/docs/front-matter/)이라는 메타데이터가 있습니다:

```text title="my-doc.md"
// highlight-start
---
id: my-doc-id
title: My document title
description: My document description
slug: /my-custom-url
---
// highlight-end

## 마크다운 제목

[links](./hello.md)를 사용한 마크다운 텍스트
```

## Links

URL 경로 또는 상대 파일 경로를 사용하는 일반 마크다운 링크가 지원됩니다.

```md
Let's see how to [Create a page](/create-a-page).
```

```md
[페이지 만들기](./create-a-page.md) 방법을 살펴봅시다.
```

**결과:** [페이지 만들기](./create-a-page.md) 방법을 살펴봅시다.

## Images

일반 마크다운 이미지가 지원됩니다.

절대 경로를 사용하여 정적 디렉토리(`static/img/docusaurus.png`)에 있는 이미지를 참조할 수 있습니다:

```md
![Alt 텍스트](/img/docusaurus.png)
```

![Docusaurus logo](/img/docusaurus.png)

현재 파일을 기준으로 이미지를 참조할 수도 있습니다. 이 기능은 이미지를 사용하여 마크다운 파일에 가깝게 이미지를 배치할 때 특히 유용합니다:

```md
![Docusaurus logo](./img/docusaurus.png)
```

## 코드 블록

마크다운 코드 블록은 문법 강조 기능을 지원합니다.

````md
```jsx title="src/components/HelloDocusaurus.js"
function HelloDocusaurus() {
  return <h1>안녕, 도큐사우르스!</h1>;
}
```
````

## 권고 사항

도쿠사우루스에는 권고 사항과 콜아웃을 만드는 특별한 문법이 있습니다:

```md
:::tip 꿀팁

멋진 꿀팁 기능을 사용하세요

:::

:::danger 주의 사항

이건 위험한 작업입니다

:::
```

:::tip 꿀팁

멋진 꿀팁 기능을 사용하세요

:::

:::danger 주의 사항

이건 위험한 작업입니다

:::

## MDX 와 React 컴포넌트

[MDX](https://mdxjs.com/)는 문서를 더욱 **대화형**으로 만들 수 있으며, 마크다운 내에서 모든 **리액트 컴포넌트**를 사용할 수 있습니다:

```jsx
export const Highlight = ({children, color}) => (
  <span
    style={{
      backgroundColor: color,
      borderRadius: '20px',
      color: '#fff',
      padding: '10px',
      cursor: 'pointer',
    }}
    onClick={() => {
      alert(`You clicked the color ${color} with label ${children}`)
    }}>
    {children}
  </span>
);

이건 <Highlight color="#25c2a0">Docusaurus 녹색</Highlight> !

이건 <Highlight color="#1877F2">Facebook 파란색</Highlight> !
```
